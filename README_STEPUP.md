# StepUp (scaffold)

This workspace contains a scaffold for `stepup-frontend` (React + Vite) and `stepup-backend` (FastAPI).

Frontend:
- Run `npm install` inside `stepup-frontend` and then `npm run dev` to start.

Backend:
- Create a Python virtualenv, install from `stepup-backend/requirements.txt` and run `uvicorn app.main:app --reload --port 8000`.
